## project

### 技术栈
* react
* redux
* immutable
* dva
* roadhog
* ant design
* less
* fetch
* mock
* react-router
* react-router-redux
* redux-auth-wrapper
* ES6/7

### 开发:

1.安装依赖
```bash
npm install
```

2.启动本地服务
```bash
npm start
```
打开 http://localhost:8000


### 打包构建:
```bash
npm run build
```
